Política de Privacidade

SistPlantão Pro - atualizado em 03/05/2026

Esta política explica como o SistPlantão Pro trata informações usadas para criar escalas, serviços avulsos, lembretes e exportações.

Dados salvos no dispositivo

O app salva localmente os compromissos cadastrados, preferências de tema, antecedência dos lembretes e registros necessários para evitar avisos duplicados. Esses dados ficam no dispositivo do usuário.

Conta e envio de dados

O SistPlantão Pro não exige login e não envia suas escalas para um servidor próprio. O app não vende dados pessoais.

Notificações

As permissões de notificação e alarme são usadas para avisar sobre compromissos criados pelo usuário. Os alertas podem ser desativados nas configurações do app ou do Android.

Arquivos exportados

PDFs, arquivos ICS e backups JSON são gerados somente quando o usuário solicita. Esses arquivos podem conter nomes de unidades, datas, horários e observações informadas pelo usuário.

Serviços de terceiros

A versão web pode carregar fontes, ícones e bibliotecas de interface/exportação por CDNs. Esses provedores podem receber informações técnicas comuns de acesso, como endereço IP e dados do navegador. A versão Android empacotada mantém os dados de escala localmente.

Retenção e exclusão

Os dados permanecem no dispositivo até que o usuário remova compromissos no app, importe um backup substituindo os dados, limpe os dados do aplicativo/navegador ou desinstale o app.

Contato

Para dúvidas de privacidade ou suporte, use o contato informado na página do aplicativo na Google Play Store.
